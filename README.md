# AI Personal Finance Manager

An AI-powered web application designed to help users manage their personal finances efficiently.  
The platform allows users to track income and expenses, scan bills, automatically categorize transactions, and analyze financial behavior through visual analytics.

This project explores how **AI and modern web technologies can improve personal financial management.**

---

## 🚀 Features

- Track income and expenses
- Expense categorization system
- Bill scanning and automatic expense detection
- AI-based transaction category prediction
- Financial analytics dashboard with charts
- Visual representation of spending patterns
- Secure user authentication
- Responsive web interface

---

## 🧾 Smart Bill Scanner

Users can upload or scan bills and the system automatically:

- Extracts bill information
- Detects transaction amount
- Identifies expense category
- Adds it to the financial dashboard

This helps reduce manual data entry and improves financial tracking.

---

## 📊 Financial Analytics

The platform provides visual insights including:

- Expense distribution charts
- Category-based spending analysis
- Financial activity overview
- Monthly spending patterns

Charts help users understand their financial habits and make better budgeting decisions.
